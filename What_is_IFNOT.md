> ⚪ **파이썬에서 `if not X:`가 True가 되는 경우 요약**

```
| 표현 (`X`)        | 의미 또는 상태            | `if not X` 결과 |
|------------------|--------------------------|-----------------|
| None             | 아무것도 없음              | ✅ True          |
| False            | 거짓                      | ✅ True          |
| 0                | 숫자 0                    | ✅ True          |
| 0.0              | 소수 0                    | ✅ True          |
| ""               | 빈 문자열                 | ✅ True          |
| []               | 빈 리스트                 | ✅ True          |
| {}               | 빈 딕셔너리               | ✅ True          |
| ()               | 빈 튜플                   | ✅ True          |
| set()            | 빈 집합                   | ✅ True          |
```

💡 `if not X:`는 X가 “비어있거나, 0이거나, False 같은 값”일 때 True가 됩니다.
