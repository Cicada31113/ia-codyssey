> ⚪ **파이썬에서 `if not X:`가 True가 되는 경우 요약**  
> 아래 값들은 모두 `if not X:` 조건에서 True로 평가됩니다.

| 표현 (`X`) | 의미 또는 상태       | `if not X` 결과 |
|------------|----------------------|------------------|
| `None`     | 아무것도 없음         | ✅ True           |
| `False`    | 거짓                 | ✅ True           |
| `0`        | 정수 0               | ✅ True           |
| `0.0`      | 소수 0               | ✅ True           |
| `""`       | 빈 문자열            | ✅ True           |
| `[]`       | 빈 리스트            | ✅ True           |
| `{}`       | 빈 딕셔너리          | ✅ True           |
| `()`       | 빈 튜플              | ✅ True           |
| `set()`    | 빈 집합              | ✅ True           |




# False 예시
```bash
name = ""  # 아무것도 입력 안 한 상태
if not name:
    print("이름이 비어 있습니다!")
